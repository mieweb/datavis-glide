*******************
Pivot Control Field
*******************

A pivot control field is the user interface that makes up a single field within the pivot control.
These can be dragged & dropped between the three different field areas of the pivot control.  They
can also be filtered.

Filtering
=========

Since a pivot table doesn't show the original data, it doesn't make much sense to filter on what is
being shown.  Instead, each pivot control field acts as like the column of a grid, and can have
multiple grid filters on it.
